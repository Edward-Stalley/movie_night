#  🍿 Movie Night

A full-stack web application that allows users to **to plan/schedule movie nights with friends by shortlisting films, voting, and keeping a record and reviews of watched films.**.

Built with **Next.js, Tailwind CSS, DaisyUI, TMDB API**, focusing on clean UX, responsive design, and smooth user interactions.

## ✨ Features

- Browse and interact with content in real-time
- Search and filter functionality
- Detailed view pages for each item
- User actions (e.g. save, vote, create, etc.)
- Responsive design (mobile-first)
- Loading, empty, and error states handled gracefully

---

## 🧭 User Flow

### 🚀 Core Flow (Main User Journey)

1. User lands on the homepage
2. Login in with OAuth
3. Browses or searches available content  
4. Clicks into a detailed view for synopsis or trailer
5. Performs main action (e.g. save, vote, interact, etc.)  
6. UI updates instantly with feedback (toast / state update)

## 🧩 Key Design Decisions

- Used **Tailwind + DaisyUI** for consistent UI system and fast development
- Followed **component-based architecture** for reusable UI elements
- Prioritized **mobile-first responsive design**
- Focused on **UX states (loading / empty / error)** to improve user experience
